Trend Heatmap

This portion of the project is looking at how color usage changed over time. This visualization is broken into three main segments: Sets, Colors, and Themes.


The first segment is looking at how sets used colors throughout time. In this data set there are over 16,000 sets, in order to have the visualization readable the number of sets have been binned based on the number of pieces, and the color usage value was the average number of unique colors that have been used by that binned segment. Over time the number of unique colors within given sets has increased over time, especially in sets that have a larger piece count.


The second segment is looking at how all of the unique colors changed over time. The y-axis is all of the unique colors that were in the data set, and the scaling value is the number of times this color was used within all sets in a given year. Overall, more colors are used as the time progresses.


The last segment of this visualization is exploring how many unique colors each theme uses in a given year. This visualization shows as time progresses so does the number of unique colors used in each theme. However, this growth is relatively minimal.
